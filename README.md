# ATM System v2 (Python OOP)

A console-based ATM simulation built using Python and Object-Oriented Programming.

## Features

* PIN authentication
* Deposit & Withdraw
* Check Balance
* Transaction History
* Change PIN

## Tech Used

* Python
* OOP (Classes, Encapsulation)

## How to Run

```bash
python 2_ATM2.py
```

## What I Learned

* Designing systems using OOP
* Handling user input & validation
* Debugging real errors during development
